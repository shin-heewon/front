- rem , em   : 글꼴의 크기 지정 단위  (상대 단위)
  px (절대단위) : 페이지 크기가 줄든, 키우든 같은 크기를 갖는다. 
  상대 단위로 설정하면, 크기가 해상도에 따라 유동적으로 변하는 장점이 있다. 
  html 요소의 기본값이 16px = 1rem 


- overflow 속성 
  내용이 요소의 크기를 벗어났을때 어떻게 처리할지를 정하는 속성 
  CSS 는 기본값 visible 속성을 가진다. 
  hidden 으로 바꿀 수 있다.  hidden으로 바꾸면, 넘친 컨텐츠가 보이지 않게 된다.   

 overflow 속성 (구글링 CSS overflow)
  - visible
  - hidden
  - scroll 
  - auto


- 배경이미지를 결정할때  url 
  background-img: url("")  ""안에 이미지 주소를 지정해서 해당 이미지를 불러서 사용한다. 
  border-radius : 요소의 테두리를 둥글게 만든다. 
  max-width : 최대 너비 설정 
  text-align : 텍스트 정렬 ( left, right, center ,justify 양쪽정렬)
