웹 브라우저는 '미디어 쿼리' 기능을 지원
미디어 쿼리란 미디어 타입에 따라 css를 다르게 적용할 수 있다.
프린터, 데스크탑, 스마트폰, 태블릿, 키오스크, 시각장애용 웹 브라우저 마다 적합한 웹페이지를 구현할 수 있게 지원한다.
    => 결론 : 다양한 장치에 따라 해상도 크기 및 비율이 제각각 다른 경우에
                미디어 쿼리를 적용한 웹 페이지는 개발효율성과 유지보수에 큰 장점이다.


2. 설정
    2-1. 
    meta 태그 : 웹 페이지에 추가 정보를 제공
    <meta name="반응형 쇼핑몰 구현" content="신세계INC 자바개발자 미니프로젝트">
    <meta name="description" content="신세계INC 프로젝트 포트폴리오"> --검색엔진 노출됨!
    => 검색시 meta 태그에 집어넣은 name이나 ㅏcontent  값이 대상이 된다. 즉 크롤링 대상이 되어 검색이 수월해진다.

    2-2. viewport
        <meta name="viewport" content="width height initial-scale=2.0(초기확대 비율)">
        viewport : 화면의 비율을 설정할 때 쓰는 메타 태그

    2-3. 미디어 쿼리
        @media(<미디어쿼리>){
            CSS 코드
        }

        <link rel="stylesheet" href="파일이름" media="미디어쿼리"> 적용

        @ : 규칙, CSS 내부에서 특정한 규칙을 표현
