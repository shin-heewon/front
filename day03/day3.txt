1. 검색엔진 : 여러 웹 사이트나 웹 페이지를 검색해주는 시스템이나 프로그램
            웹에서 원하는 정보를 찾는 시간을 최소화해주는 프로그램 

2. 검색 엔진 동작원리 
   사용자가 검색을 했을때, 원하는 결과를 빠르게 보여주려면
   수많은 정보를 미리 정리해 놓아야 하는데, 
   이 과정을 자동으로 데이터베이스화 시켜주는 프로그램을 bot(봇)  또는 크롤러(crawler) 

   검색엔진 봇이 주기적으로 웹 사이트들을 방문해서 쌓은 정보를 정리하여, 데이터베이스에 저장한다. 
   해당 정보를 사용자가 검색할 때 결과를 빠르고 정확하게 보여준다. 


3. CSS  
   웹페이지의 요소를 시각적으로 꾸미는 작업 기술 

   CSS의 장점  재사용성  및 반복작업이 줄어듬 

4. 선택자   (selector)
   1. 전체선택자   * 
   2. 유형 선택자 
   3. 아이디 선택자
   4. 클래스 선택자
   5. 복합 선택자 
   6. 속성 선택자 []
   7. 자손 선택자 >
   8. 반응 선택자  :active :hover  사용자의 반응으로 생성되는 특정한 상태를 선택할때 사용함 
   9. 상태 선택자  :ckecked  :focus :enabled   :disabled

   10. 구조 선택자  : 특정 위치에 있는 태그를 선택할때 사용  
                      :first-child  형제관계에서 첫번째로 등장하는 태그 선택
                      :last-child                마지막으로 등장하는 태그 선택
                      :nth-child            앞에서 수열 몇 번째 지정하여 선택
                      :nth-last-child       뒤에서 수열 몇 번째 지정하여 선택 

11. CSS 배치 속성  : CSS 레이아웃 
    박스 모델 : 마진margin, 테두리 border , 패딩 padding 
    반응형 웹 페이지  : 기기 화면에 맞춰 요소가 자동으로 배치된 웹 페이지 


    Block level 요소  : h1    : ㅣ요소가 있는 가로 줄 전체를 차지한다.
    inline level 요소 : span  :  요소가 있는 공간만을 차지한다.  

    HTML 모든 요소는 display 속성 이 있다. 


12. 반응형 웹페이지  : 화면 해상도에 따라서 HTML과 CSS 가 자동을 맞춰서 보여주는 기능을 제공하는 웹페이지 
    미디어쿼리를 이용해서 제공 
    미디어쿼리란? 사이트에 접속하는 미디어 타압과 특징에 따라 다른 CSS 스타일을 적용하도록 도와주는 규격 
   
     @media (color) { .... }
     @import url("hello.css")
     @import 'hello.css' ;       
     주의할 점  @import 는 페이지의 최상단에 기술해야 한다.

     13. CSS 꽃   : 레이아웃 
         layout  :   이미지, 텍스트 내가 원하는 위치로 배치하는 것 

         - 플렉스박스 (flexbox) - 유연한 
           정렬 (align)
           축 바꾸기  : flex-direction 속성 

           원칙 1. 부모만이 플렉스 할 수 있다.  

           body { 
               display : block, inline, flow, grid,flex;
           }
           자식 요소를 원하는 방향(정렬, 축 이동)으로 유연하게 배치할 수 있다. 

           주축(가로) 과 교차(세로)축     =>  flex-direction 속성을 이용하여 축을 변경가능하다. 
           justify-content 속성  : 웹 페이지의 축을 따라 요소 사이사이에 공간을 만드는 속성 


           교차축 정렬하기 : align-items  속성 : 교차축 정렬
